# Introdução

![](https://imgur.com/dsR2CVK.gif)

Este projeto é um gerador de diamantes em C#. O programa solicita ao usuário um número ímpar maior que 1 e, em seguida, exibe um diamante na tela usando o caractere 'X'.

## Passo a Passo

- Cria um diamante X com base nos dados recebidos
- Organizado com static void para cada parte do código
- Capaz de continuar ou sair do jogo com opção de saída (S/N)
